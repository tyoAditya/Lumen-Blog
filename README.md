# Lumen Blog

## Used in this project:
* Laravel Lumen 8
* Passport OAuth

## Available Scripts To Run In Order
* Firstly, create a mysql database and create .env file from .env.example

```bash 
$ composer install
```
```bash 
$ php artisan key:generate
```
```bash 
$ php artisan migrate --seed
```
```bash 
$ php artisan passport:install
```
```bash 
$ php artisan serve
```
